What is pyATS ? pyATS is an essential tool in the NetDevOps toolbox that helps streamline and standardize network operations and testing.

I got familiarized with PyATS core components which are 
- The foundational platform: it’s the original pyATS framework
- Library (Genie): it’s the platform vendor-agnostic library
-	Web dashboard(Xpresso): it’s a beautiful web dashboard to manage test suites,testbeds, test results, and their insights.

Use cases of pyATS
- Network Health Check Automation
- Network Troubleshooting and Debugging
- Network Test Automation
